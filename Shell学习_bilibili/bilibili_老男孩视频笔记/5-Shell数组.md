## 二 基本数组

### 2.1 数组语法

> NAME=(元素0 元素1 元素2 元素3 ...... )

### 2.2 数组读出

> \${ARRAY[i]}

### 2.3 数组赋值

* 一次赋一个值

  A[0]='a'

  A[1]='b'

* 一次多个值

  A=('a' 'b' 'c')

  B=('cat /etc/passwd') #将该文件中的每一个行作为一个元素赋值给数组

  C=(\`ls /var/ftp/Shell/for\`)

### 2.4 查看数组

#declare -a 	#查看系统中声明过的数组

### 2.5 访问数组元素

```shell
echo ${arr[0]}
echo ${arr[@]}#查看数组所有的元素 == echo ${arr[*]}
echo ${#arr[@]}#统计数组元素个数
echo ${!arr[@]1:3}#从下标1开始访问3个，1，2，3
```

## 三 关联数组

### 3.1 声明一个关联数组

```shell
declare -A arr
acc=([name]='ly' [age]='18')
acc[test]='aa'
```

关联数组可以自定义索引，感觉有点像字典，要声明为关联数组。
